Tags

  • h1 a h6
    • A tag "h1" é utlizada para Título e vai até o nível "h6".


  • ol
    • A tag "ol" é utilizada para lista ordenada.


  • ul
    • A tag "ul" é utilizada para lista não ordenada.


  • li
    • A tag "li" é utilizada para lista com ponto de marcação.


  • br
    • A tag "br" é utilizada para quebra de linha.


  • div
    • A tag "div" é utilizada para bloco de código.


  • p
    • A tag "p" é utilizada para parágrafo.


  • img scr
    • A tag "img scr" é utilizada para inserir imagens.


  • b
    • A tag "b" deixa o texto em negrito.


  • strong
    • A tag "strong" deixa o texto em negrito, mas dá uma intonação diferente para os deficientes visuais na leitura do texto.


  • a
    • A tag "a" ancora o texto aalgum outro conteúdo.
